Detailed Description

BookingsConfirmedV

Contains interaction logic for BookingsConfirmedV.xaml From this view user can access the list of future confirmed bokings, starting from today. Filtering by ID, event date, customer name and room name is enabled. Clicking on single row (booking) displays booking details on bottom half of the page. Each booking may be modified by user with buttons "Muuda" - change/update booking details, and "Kustuta" - delete booking. Deleting booking requires additional confirmation by clicking OK in MessageBox.
