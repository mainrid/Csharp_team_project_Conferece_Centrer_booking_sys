BookingUpdateV

Contains interaction logic for BookingUpdatedV.xaml From this view user can create new Booking or modify details for existing booking. In case of new booking the firlds are empty In case if booking update fields are pre-filled with booking data Customer can be either chosen from database, or created New. Several input validation methods ensure correct information is stored in DB
